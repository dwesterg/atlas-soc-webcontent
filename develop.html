Develop on your PC

 

Whether you are a hardware or software developer, this page contains the tools and resources you need to begin development on your own PC. Look in the Design Resources section for links to example designs and accessory boards you can use with the Atlas-SoC kit.

 

 

  • Set Up Your Software Development Environment

    Download and install the software development tools and documentations to your PC or workstation from one of the following sources.

     

    Linux

    Bare Metal or Other OS/RTOS

    Repository

     

    Links to the github repository are available on Rocketboards.org to get the latest Linux distribution.

    Developing bare-metal applications requires the appropriate bare-metal compilers that can generate the correct binaries for the target device. To setup the environment, you need to install the latest SoC Embedded Development Suite.

     

    Detail & download link

     

    The full installation includes the following components:

     

    • Compiler
      • GCC toolchain with NEON and floating point support

     

    • ARM DS-5 Altera Edition
      • ARM Compiler5 with NEON and floating point support (30 day evaluation)
      • FGPA-adaptive debug

     

    • Hardware Libraries (HWLibs)
      • Provides easy access to low-level hardware for configuration and control

     

  • Set Up Your Hardware Development Environment

    Download and install the Quartus Design Software to create and modify SoC hardware designs. With Quartus you can develop hardware for the FPGA, configure the HPS (I/O assignments, clocks, etc.), program and debug the FPGA hardware.

     

    What You Need

     

    • Quartus II Web Edition (free)

    or

    • Quartus II Subscription Edition (paid license)

     

    • Golden System Reference Design (GSRD) for the Atlas-SoC board

    MicroSD Requirements

    • Capacity: 4GB minimum
    • Speed: Class 4 (minimum)

     

    Programming:

    To program a microSD card Linux image on a Windows PC you can use a free tool called Win32DiskImager.exe from

    http://sourceforge.net/projects/win32diskimager/.

     

    The steps below present how to create microSD card on a windows machine using Win32DiskImager.exe.

    1. Connect the microSD card to a Windows PC
    2. Execute Win32DiskImager.exe
    3. Select the image file for microSD card
    4. Select the microSD card device
    5. Click “write” to start writing the image file to the microSD card. Wait until the image is written successfully.
