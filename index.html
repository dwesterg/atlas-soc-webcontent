Welcome to the Atlas-SoC Software Development Platform!

 

This kit will help you learn about Altera® SoC devices which combine the performance of a dual-core ARM® CortexTM-A9 MPCoreTM processor with the flexibility of programmable logic. This new class of device opens exciting new possibilities for the embedded developer. The following web pages will guide you through the steps of learning about the board from evaluation to full development. Here are some things you can do with this kit:

 

Play

 

Interact with the board using a web-based interface.

Learn

 

Access technical information and resource links to quickly learn the fundamentals of SoC development.

Try

 

Try it for yourself. Get hands-on experience developing embedded software using the Atlas-SoC board as your development system.

Develop

Download and install the development tools on your PC or workstation. Learn about daughter cards, open source projects, and additional technical information available specifically for this board.

    SoC Device Configuration:

     

    FPGA Hardware

    • 32-bit FFT
    • x2 DMA
    • 4k SRAM buffer
    • GPIO for LEDs, push buttons & slide switches

     

    HPS I/O

    • 32-bit DDR3 SDRAM
    • x1 10/100/1000 Ethernet port
    • x1 USB 2.0 OTG port
    • SD Card I/F
    • GPIO for LEDs & push buttons
    • x1 UART for stdio
    • x1 I2C for accelerometer

    OS: Linux
